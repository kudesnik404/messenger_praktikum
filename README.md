# Messenger SPA

Учебный проект мессенджера в формате Single Page Application.

## Стек

* TypeScript
* SCSS
* Vite
* Handlebars
* ESLint
* Stylelint

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run start
```

Проект будет доступен по адресу:

```text
http://localhost:3000
```

## Сборка проекта

```bash
npm run build
```
