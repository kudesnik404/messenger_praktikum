# Messenger SPA

Учебный проект мессенджера в формате Single Page Application.

## Стек

* TypeScript
* SCSS
* Vite
* Handlebars
* ESLint
* Stylelint

## Установка

```bash id="m2whv4"
npm install
```

## Запуск проекта

```bash id="5y6s2d"
npm run start
```

Проект будет доступен по адресу:

```text id="q9f7x1"
http://localhost:3000
```

## Сборка проекта

```bash id="r0v2gn"
npm run build
```

## Страницы

* Чаты — `/#/`
* Вход — `/#/login`
* Регистрация — `/#/register`
* Профиль — `/#/profile`
* Ошибка 404 — `/#/404`
* Ошибка 500 — `/#/500`
